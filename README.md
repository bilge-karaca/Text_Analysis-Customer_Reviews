# Text_Analysis-Customer_Reviews
This project contains the customer review analysis of a product on an e-commerce website. 

The dataset contains **4977 reviews**. I scraped the data through the application I created. Check out here: <https://github.com/bilge-karaca/Customer_Review_Scraping>
